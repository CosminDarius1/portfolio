Caterpillar Countdown

During this project I tested my JavaScript skills and along with my teammates we created Caterpillar Countdown which is a JavaScript based computer game where the user can test its mental agility.The game consists of rounds and within each round the user will see 3 to 5 different numbers between 0-100.The user needs to click on the numbers in ascending order to allow to progress to the next round. Each round that is completed will add 1 point to the score and a timer set for 30 seconds will start when the user will click on play. For more details please check GitHub repository
